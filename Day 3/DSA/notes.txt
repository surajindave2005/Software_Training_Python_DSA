Time complexity
    -> Depends of condition 
    -> most used Big O - O(N) -> BEST CASE
                 Big-Omega : AVERAGE CASE
                 Big-Theta :  WORST CASE
    

Constant time -> O(1)
    -   Accessibg a specific elements in arrray.

Linear Time -> O(N)
    -   Loop through arraye elements.
    -   example
        for elements in array:
            print(elements)

Logarithmic Time -> O(LogN)
    -   Find an elements in sorted array.
    -   only some index visited
    -   Binary search algo 
    -   example:
        arr = [1,2,3,4,6]
        for index in range(0, len(array), 3):
            print(arr[index])
    

Quadradic Time -> O(N sqr);
    -> WORST CASE
    -> Looking at every index twice

        print(x, y)
